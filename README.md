QR Code Generate

This repository contains the QR Code Generate project, a web application developed to generate QR codes dynamically based on user input.

Live version:
https://lucashenrique1908.github.io/Qr-Code-Generate/

Description

QR Code Generate is a simple web app that allows users to enter text or a URL and receive a corresponding QR code instantly. It provides a clean interface for quick QR code creation without the need for external tools.

This project was built primarily for practice and learning purposes with front-end technologies.

Technologies Used

* HTML

* CSS

* JavaScript

Purpose

The main goals of this project are to:

 - Practice JavaScript logic and event handling

 - Work with dynamic DOM updates

Build a functional and responsive user interface

The application focuses on interactive functionality and basic UI responsiveness.

How to Run Locally

Clone the repository: git clone https://github.com/lucashenrique1908/Qr-Code-Generate.git

Notes

This project demonstrates foundational front-end skills and is intended for education and portfolio use.
